# Class Repository

This repository collects course materials for the class:

- Syllabus and course policies
- Weekly reading lists
- Lecture slides
- Public links and references

## Repository Layout

```text
.
├── syllabus/
│   └── syllabus.md
├── readings/
│   └── README.md
├── slides/
│   └── README.md
└── resources/
    └── sharing-guidelines.md
```

## For Students

Start with the syllabus, then use the readings and slides folders for weekly materials.

## For Instructors

Add readings as links or citation entries whenever possible. If you need to distribute files directly, make sure you have permission to share them through this repository.
