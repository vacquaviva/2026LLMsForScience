# Slides

Use this folder for lecture slides.

Recommended pattern:

```text
slides/
├── README.md
├── week-01.pdf
├── week-02.pdf
└── week-03.pdf
```

PDF is usually the easiest format for students to view across devices. If you also want to keep editable source files, consider using a matching filename such as `week-01.pptx` or `week-01.key`.

When adding slides, use consistent filenames:

- `week-01.pdf`
- `week-01.pptx`
- `week-02.pdf`
- `week-02.pptx`
