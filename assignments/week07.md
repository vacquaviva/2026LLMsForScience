# Assignment: Evaluating Literature Search Tools

This assignment is a continuation of our in-class exercise. We compare different LLM tools for reference search, and consider (at least) at Asta (Allen AI), Google Scholar Labs, and SciSpace. If you work in Astronomy, you can also use Pathfinder.

## Task

Run a search on a topic familiar enough that you’ll be able to evaluate the results. For example, I ran the following prompt: "I need a set of papers to learn the basics about the ocean carbon cycle, suitable for advanced undergraduate or beginning graduate students".

## Report

Report your qualitative assessment of the quality of the lists, in particular looking at 1) Relevance of results; 2) Missed seminal references; 3) Evident biases (for example, towards books vs papers, famous/less known, old/new); 4) 
What other tools are offered (summaries, reference formatting, follow-up questions, attribution of content within references).

Also report your final pick (if any).
