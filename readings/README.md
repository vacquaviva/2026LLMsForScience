# Readings

Use this folder to organize weekly readings.

Recommended pattern:

```text
readings/
├── README.md
├── week-01.md
├── week-02.md
└── week-03.md
```

Prefer stable links, library permalinks, or full citations over uploaded copies unless you have permission to distribute the files.

## Reading Template

```markdown
# Week 01 Readings

## Required

- Author, "Title," publication, year. Link:

## Optional

- Author, "Title," publication, year. Link:
```
